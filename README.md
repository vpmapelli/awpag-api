# About

This is a simple study Spring Restful API project  developed during an online event called "Intensivão Spring REST" (Spring REST Intensive week, in free translation) organized by Algaworks.

# Topics studied

- Spring ecosystem overview
- Repository resources
- HTTP codes and status
- Content negotiation
- Flyway
- Jakarta Persistance
- Spring Data JPA
- CRUD operations
- Jakart Bean Validation
- Domain services
- Validation groups
- Best practices employing Exception Handler
- Domain model and representation model isolation
- Employing ModelMapper for transforming objects from distinct app layers

# How to execute

Simplest way to execute this api is navigating to project root folder and running:

```
docker compose up
```

Make sure you are using [Docker Compose V2](https://docs.docker.com/compose/migrate/).

After executing this command, the api is available at localhost:8080.

This has been tested only on Linux OS, specifically Ubuntu 22.04.
